# stock - iOS

iOS project for stock gyo app.

## Requirements

- **xtool**: Cross-platform iOS development tool (works on Linux, macOS, Windows)
- **iOS 14.0+**

## Project Structure

This project uses **Swift Package Manager (SwiftPM)** with **xtool** for cross-platform iOS development:
- Version control friendly
- Works on Linux, macOS, and Windows
- No Xcode required

## Setup

1. Install xtool:
   ```bash
   curl -L https://xtool.sh/install.sh | bash
   ```

2. Set up xtool (first time only):
   ```bash
   xtool setup
   ```

## Development

### Build and Run
```bash
# Using gyo CLI (recommended)
gyo run ios

# Or use xtool directly
xtool dev
```

The app will automatically connect to the development server URL configured by gyo.

## URL Configuration

The app loads content from the URL specified in `Sources/Resources/gyo-config.json`:

```json
{
  "serverUrl": "http://192.168.0.7:3000"
}
```

This file is automatically generated and updated by `gyo run ios` with the correct server URL.

## Files

- `Package.swift` - Swift Package Manager configuration
- `xtool.yml` - xtool configuration (bundle ID, etc.)
- `Sources/App.swift` - SwiftUI app entry point
- `Sources/WebViewContainer.swift` - WebView wrapper with gyo runtime
- `Sources/Config.swift` - Configuration loader
- `Sources/Resources/` - App resources (storyboards, config files)

## Notes

- xtool allows iOS development on Linux without macOS/Xcode
- The app uses SwiftUI and WKWebView to display web content
- Configuration is managed via JSON files
- xtool homepage: https://xtool.sh
